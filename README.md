# Stroke Trials Screener Public Demo

A mobile-first Progressive Web App for demonstrating stroke-trial eligibility logic. The public GitHub Pages deployment is synthetic/demo-only.

## Public-Use Boundary

- Do not enter real patient identifiers, PHI, research participant information, or operational clinical data.
- URL-hash patient prefill is disabled on the public GitHub Pages deployment.
- The public site is not an official system, recruitment workflow, trial source of truth, or bedside clinical tool.
- Verify current trial status, eligibility criteria, and contact workflow against study-owner-approved and IRB/HSD-approved materials before any clinical or recruitment action.

## Features

1. **Eligibility logic demo:** Interface to test de-identified or synthetic parameters such as onset/LKW, NIHSS, ASPECTS, and pre-mRS against trial criteria.
2. **PWA support:** Installable as an app on iOS or Android for offline demonstration use.
3. **Offline support:** Supports offline use after first load once resources are cached by the service worker.

## Mobile Installation

- **iOS (Safari):** Open the website, tap the Share button, scroll down, and select Add to Home Screen.
- **Android (Chrome):** Open the website, click the top-right menu icon, and tap Add to Home screen or click the Install App button in the header.
